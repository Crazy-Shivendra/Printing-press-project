# Online Printing Press – Custom Print Ordering System

This project allows users to customize and order printing materials such as:
- Wedding Cards
- Flex Banners
- Posters
- Bill Books
- Custom Notebooks
- Writing Pads

## Features
- Template selection (Hindu / Muslim / Sikh / Any)
- Live text customization & preview
- Quantity-based pricing with discounts
- WhatsApp Demo Request
- Simple Admin Panel
